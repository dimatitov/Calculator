### `Рабочий калькулятор`

### `npm install`
Установка модулей create-react-app,
typesript`a, tslint, stylelint

### `npm start`
Запуск проекта в браузре через host

### `npm test`
Запуск тестов

### `npm run build`
Создание финального билда

### `npm run lint-ts`
Запуск анализа кода typescript

### `npm run lint-styles`
Запуск анализа стилей css
